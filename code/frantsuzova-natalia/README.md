## Задание

Разработать программу, которая позволяет выполнять:
 - расчет цены как основного параметра,характеризующего рыночное равновесие
 - просмотр,какие неценовые факторы влияют на спрос
 - просмотр,какие факторы влияют на предложение
 - количественное вычисление эластичности спроса
 - количественное вычисление эластичности предложения

Все необходимые математические сведения можно найти в статье
["Закон спроса и предложения"][Supply and demand] на сайте Wikipedia.
