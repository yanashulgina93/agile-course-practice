Выполнил:

## Задание

Простое приложение для учета личных финансов. Необходимо реализовать:

 - Представление данных в виде таблицы
 - Сохранение и загрузка данных
 - Добавление:
   + Категория
   + Наименование
   + Количество
   + Дата
   + Сумма расхода
   + Сумма дохода
