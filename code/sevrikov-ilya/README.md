# Операции над 3D векторами

Выполнил:

 - Севриков Илья
 - ННГУ, ИИТММ, каф. МОСТ, группа М0823-1

## Задание

Реализовать класс, предназначенный для работы с векторами в трехмерном пространстве.
Данный класс должен поддерживать следующие операции:

 - Вычисление нормы.
 - Нормализация.
 - Скалярное и векторное произведения.

Все необходимые сведения можно найти в статьях:
 - ["Норма"][norm] на сайте Wikipedia
 - ["Скалярное произведение"][dotproduct] на сайте Wikipedia
 - ["Векторное произведение"][crossproduct] на сайте Wikipedia
